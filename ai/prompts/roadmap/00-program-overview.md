# Program overview

## Theme

Budget V2 aims to make budget creation faster, guided, and increasingly connected to imported transactions.

## Three capability layers

1. Guided entry into budgeting
2. Assisted transaction categorization
3. Budget generation and distribution from historical operations

## Recommended implementation order

1. Add a strong entry point on the Budget page
2. Add the questionnaire-based wizard
3. Add a manual path with better guardrails
4. Add categorization confidence and user validation
5. Add budget bootstrap from validated imported history
