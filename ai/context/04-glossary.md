# Glossary

- **Budget Wizard** : flow guidé pour créer ou initialiser un budget.
- **Manual Mode** : saisie directe par l'utilisateur.
- **Generated Mode** : budget proposé à partir d'un questionnaire ou d'un historique importé.
- **Confidence Score** : score de confiance associé à une suggestion de catégorisation.
- **Pre-reconciliation validation** : validation utilisateur de la catégorie avant pointage.
- **Budget bootstrap** : initialisation d'un budget à partir de données existantes.
