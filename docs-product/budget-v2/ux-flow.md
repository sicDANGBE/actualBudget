# UX flow

## Entry

Sur la page Budget, afficher une entrée claire vers Budget V2.

## Path A - Guided questionnaire

1. Revenus
2. Charges fixes
3. Aspirations
4. Épargne
5. Synthèse
6. Appliquer

## Path B - Generated from imports

1. Import ou sélection d'historique
2. Suggestions de catégories
3. Validation selon score de confiance
4. Proposition de budget
5. Synthèse
6. Appliquer

## Path C - Manual

1. Entrer en mode manuel
2. Saisir / ajuster catégories et montants
3. Garder les garde-fous utiles
