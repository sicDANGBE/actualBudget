# Acceptance criteria

## Entry point

- L'utilisateur voit une entrée claire sur la page Budget.
- L'entrée respecte la charte Actual.

## Guided mode

- Un budget initial peut être généré via questionnaire.
- Le budget généré s'applique au système existant.

## Import-assisted mode

- Les opérations importées reçoivent une catégorie suggérée quand possible.
- Un score de confiance est disponible.
- Les cas incertains demandent une validation utilisateur.

## Safety

- Pas de moteur budgétaire parallèle.
- Pas de régression notable sur les flows existants.
